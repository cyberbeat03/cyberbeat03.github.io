Below is a list of some of the most common WinMix Desktop shortcut keys. Note that this is not a complete list.

Shortcut Command
f1 Help
F4 Display the About dialog box
Alt+F4 Exits the program
F7 Toggle Muting of audio on and off
F8 Turn volume up
F9 Turn volume down
Shift+F10 or Applications key Brings up a context menu of commands for the current playlist
DELETE Remove selected item from playlist
Shift+DELETE Removes selected playlist entry from music library
Ctrl+DELETE Removes selected playlist
ENTER Begin playback of selected item in the playlist
ESCAPE Stops playback of currently playing item
TAB Cycles forward between the controls on main screen
Shift+TAB Cycles backwards between the controls on main screen
Ctrl+c Copies the selected playlist item to the clipboard
Ctrl+Shift+C Copy all items in playlist to the clipboard
Ctrl+D Moves selected item down in the playlist
Ctrl+U Moves selected item up in playlist
Ctrl+N Create a new playlist
Ctrl+O Load audio files to current playlist
Ctrl+Shift+O Open existing playlist
Ctrl+P Toggles between play and pause of the current item
Ctrl+S or ESCAPE Stops playback of currently playing item
Ctrl+F Moves forward to next item in the playlist
Ctrl+Shift+F Fast forwards through currently playing item
Ctrl+B Moves back to previous item in the playlist
Ctrl+Shift+S Slows down the currently playing audio
Ctrl+Shift+G Speeds up currently playing audio
Ctrl+Shift+N Returns audio playing speed to normal
Ctrl+L Turns loop playback on / off
