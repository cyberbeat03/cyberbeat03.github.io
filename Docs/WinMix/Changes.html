This document lists notable bug fixes made to WinMix Desktop.

Changes In Version 4.5

  • The name has been changed to WinMix Desktop and is fully compatible with Windows 10 and the .NET framework V4.8
  • Only the name of the current playlist is displayed in the title bar along with the app name. There is a new deticated text box for displaying the current item's name and length.
  • Press enter on an entry in the playlist to make it the current media item and begin playing. Press escape to stop playback.
  • When changes are made to the current playlist they are now saved automatically.
  • When you add a media file to the playlist it automatically gets added and/or updated in your media library
  • Fixed issue of media not always displaying the most up-to-date name and length.
  • All commands for the current playlist have been moved to a context menu. When focus is on the list, simply press the applications key or shift+F10 for a menu containing available playlist commands.
  • Some keyboard shortcuts have been added or modified.
  • Other code optimization has been performed making the app run even faster and more stable.
